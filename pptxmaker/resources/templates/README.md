# PPT 模板使用指南

## 添加模板

将你的 `.pptx` 模板文件放入此目录即可。

## 命名规范

- 使用小写字母和连字符：`company-brand.pptx`
- 避免空格和特殊字符
- 文件名即模板名（不含 `.pptx` 后缀）

## 使用模板

告诉 AI 助手使用指定模板创建 PPT，例如：

- "用 blank 模板帮我做一个 2024 年度总结 PPT"
- "基于 company-2026 模板，创建季度业绩汇报"

## 如何制作企业模板（母版设置）

为了让 AI 助手能够准确识别并应用"封面页"、"结束页"和"分页标题页"，你需要在 PowerPoint 中按规范设置**幻灯片母版**：

1. **打开母版视图**：在 PowerPoint 中点击 `视图` -> `幻灯片母版`
2. **重命名版式**：右键点击左侧的缩略图，选择"重命名版式"。AI 助手依赖以下**版式名称（Layout Name）**来识别特定页面：
   - 封面页：重命名为 `Title Slide` 或 `封面`
   - 分页标题页：重命名为 `Section Header`、`过渡页` 或 `章节`
   - 结束页：重命名为 `Ending`、`结束` 或 `封底`
   - 标准内容页：重命名为 `Title and Content` 或 `内容`
3. **设置占位符**：
   - 在版式中使用 `插入占位符` -> `文本`（**不要**直接画普通文本框）。
   - 为占位符设置好字体（中文使用 `圆体-简`，英文数字使用 `effra`）、颜色（使用品牌色）、字号、行距。
   - AI 助手只能把文字填入"占位符"中，无法修改普通文本框。
4. **锁定背景**：将 Logo、背景图片、品牌色块等固定元素直接绘制在母版页上，这样 AI 助手生成时就绝对不会破坏这些品牌元素。
5. **保存**：关闭母版视图，删除所有普通幻灯片（保持文件为空），保存为 `.pptx` 文件放入此目录。

## 默认模板

- `blank.pptx` — 默认空白模板，含基本母版布局，使用 bank_brand 配色
