# 预设布局参考

> 8 种常用幻灯片布局的精确坐标参数和代码示例。
> 所有坐标使用 `Inches()` 单位，基于 16:9（13.333×7.5 英寸）画布。

---

## 1. 标题页 (title_slide)

适用：封面页、演示文稿首页

```
┌──────────────────────────────────┐
│          [全屏背景色]             │
│                                  │
│        ┌──────────────┐          │
│        │   标题文字    │          │
│        └──────────────┘          │
│        ┌──────────────┐          │
│        │   副标题      │          │
│        └──────────────┘          │
│                                  │
└──────────────────────────────────┘
```

| 元素 | left | top | width | height |
|------|------|-----|-------|--------|
| 背景 | 0 | 0 | 13.333 | 7.5 |
| 标题 | 1.0 | 2.0 | 11.333 | 1.5 |
| 副标题 | 2.0 | 3.8 | 9.333 | 1.2 |

```python
helper.add_title_slide("演示标题", "副标题文字")
```

## 2. 内容页 (content_with_title)

适用：标准要点页，最常用的布局

```
┌──────────────────────────────────┐
│ ██████████ 标题栏 ██████████████ │
├──────────────────────────────────┤
│                                  │
│  • 要点一                        │
│  • 要点二                        │
│  • 要点三                        │
│  • 要点四                        │
│                                  │
└──────────────────────────────────┘
```

| 元素 | left | top | width | height |
|------|------|-----|-------|--------|
| 标题栏背景 | 0 | 0 | 13.333 | 1.3 |
| 标题文字 | 0.7 | 0.4 | 11.933 | 0.8 |
| 正文区域 | 0.7 | 1.5 | 11.933 | 5.5 |

```python
helper.add_content_slide("页面标题", ["要点一", "要点二", "要点三"])
```

## 3. 两栏页 (two_column)

适用：优劣对比、并列展示

```
┌──────────────────────────────────┐
│ ██████████ 标题栏 ██████████████ │
├────────────────┬─────────────────┤
│  左栏标题      │   右栏标题       │
│  • 左1         │   • 右1          │
│  • 左2         │   • 右2          │
│  • 左3         │   • 右3          │
│                │                  │
└────────────────┴─────────────────┘
```

| 元素 | left | top | width | height |
|------|------|-----|-------|--------|
| 标题栏背景 | 0 | 0 | 13.333 | 1.3 |
| 标题文字 | 0.7 | 0.4 | 11.933 | 0.8 |
| 左栏 | 0.7 | 1.5 | 5.767 | 5.5 |
| 右栏 | 6.867 | 1.5 | 5.767 | 5.5 |

```python
helper.add_two_column_slide(
    "对比标题",
    "优势", ["可编辑", "轻量"],
    "劣势", ["API复杂", "无预览"]
)
```

## 6. 图表页 (chart_slide)

适用：数据可视化、统计展示

```
┌──────────────────────────────────┐
│ ██████████ 标题栏 ██████████████ │
├──────────────────────────────────┤
│   ┌──────────────────────────┐   │
│   │                          │   │
│   │      图表区域             │   │
│   │      (柱状图/饼图/折线)   │   │
│   │                          │   │
│   └──────────────────────────┘   │
└──────────────────────────────────┘
```

| 元素 | left | top | width | height |
|------|------|-----|-------|--------|
| 标题栏背景 | 0 | 0 | 13.333 | 1.3 |
| 标题文字 | 0.7 | 0.4 | 11.933 | 0.8 |
| 图表 | 1.0 | 1.5 | 11.333 | 5.5 |

```python
helper.add_chart_slide(
    "季度收入", "bar",
    ["Q1", "Q2", "Q3", "Q4"],
    {"收入": [100, 200, 300, 400]}
)
```

## 7. 总结页 (summary_slide)

适用：关键要点、结论

```
┌──────────────────────────────────┐
│ ██████████ 强调色标题 ██████████ │
├──────────────────────────────────┤
│                                  │
│  ✓ 关键要点 1                    │
│  ✓ 关键要点 2                    │
│  ✓ 关键要点 3                    │
│                                  │
└──────────────────────────────────┘
```

与内容页坐标相同，但标题使用强调色背景，要点使用 ✓ 前缀和加粗样式。

```python
helper.add_summary_slide("关键要点", ["要点1", "要点2", "要点3"])
```

## 8. 空白页

适用：自定义布局

```python
slide = helper._add_blank_slide()
# 自由添加元素
txBox = slide.shapes.add_textbox(Inches(1), Inches(1), Inches(8), Inches(1))
```
